01 Erste Schritte & Anmeldung

Thronburger POS läuft vollständig offline auf dem Kassen-PC. Es wird kein Internet benötigt — alle Bestellungen, das Menü und die Berichte werden direkt auf dem Gerät gespeichert.

Anmelden

  1. Starten Sie das Programm über das Thronburger POS-Symbol auf dem Desktop.
  2. Der Personal-Login erscheint. Geben Sie Ihre 4-stellige PIN ein (PIN eingeben).
  3. Bei falscher Eingabe erscheint Falsche PIN — einfach erneut versuchen.

⚠ Wichtig — zuerst erledigen

Das System wird mit einem Standard-Administrator ausgeliefert:
Name Admin · PIN 1234.

Melden Sie sich einmal mit dieser PIN an und ändern Sie sie sofort unter Personal → PIN ändern (siehe Abschnitt 7). So kann niemand Fremdes an Berichte oder Stornierungen.

Zwei Rollen

RolleDarf
Admin (Inhaber) Alles: Bestellungen, Berichte aller Tage, Menü, Personal, Einstellungen, Stornierungen.
Kassierer Bestellungen aufnehmen & drucken. Berichte nur für heute. Keine Stornierung ohne Admin-PIN.

Sprache umstellen

Die Oberfläche gibt es auf Deutsch, Englisch und Türkisch. Die Standardsprache stellen Sie unter Einstellungen → Standardsprache ein (siehe Abschnitt 8).

02 Kasse — eine Bestellung aufnehmen

Die Kasse ist der Hauptbildschirm. Links sehen Sie das Menü nach Kategorien (Rind, Smash, Hähnchen, Hot Dogs, Fingers, Getränke), rechts den Warenkorb.

  1. Bestellart wählen. Oben rechts: Vor Ort, Zum Mitnehmen oder Lieferung.
    • Bei Vor Ort erscheint das Feld Tisch-Nr. — Tischnummer eintragen.
    • Bei Zum Mitnehmen / Lieferung genügt der Kundenname (bei Lieferung zusätzlich die Telefonnummer). Mehr Kundendaten sind nicht nötig.
  2. Artikel hinzufügen. Tippen Sie einen Artikel an — er landet im Warenkorb. Mehrfach tippen erhöht die Menge. Solange nichts ausgewählt ist, steht dort Warenkorb leer — Artikel antippen.
  3. Bei Bedarf anpassen. Zutaten entfernen oder bezahlte Extras hinzufügen — siehe Abschnitt 3.
  4. Notiz hinzufügen (optional). Über Notiz einen Hinweis für die Küche eintragen, z. B. „gut durch". Dieser Text erscheint auf dem Küchenzettel.
  5. Rabatt (optional). Im Feld Rabatt einen Betrag abziehen. Zwischensumme und Gesamt aktualisieren sich sofort.
  6. Abschließen. Auf Abschließen & Drucken tippen, dann bei Zahlung wählen Bar oder Karte auswählen.

Nach dem Abschließen erscheint kurz Bestellung #… gespeichert und es werden automatisch zwei Belege gedruckt:

  • Küchenzettel — geht an den Küchendrucker. Nur Artikel, Mengen, entfernte Zutaten, Extras und Ihre Notiz. Ohne Preise, damit die Küche schnell arbeiten kann.
  • Kundenbeleg — geht an den Kassendrucker. Mit allen Preisen, Rabatt, Gesamtsumme und Zahlungsart.

✓ Gut zu wissen

Sind Küchen- und Kassendrucker gleich (nur ein Drucker im Betrieb), werden beide Belege einfach nacheinander auf demselben Gerät gedruckt.

03 Zutaten entfernen & Extras hinzufügen

Wenn ein Gast etwas ohne eine Zutat oder mit einer bezahlten Zutat möchte, tippen Sie im Warenkorb beim Artikel auf Anpassen.

Zutaten weglassen (kostenlos)

Unter Zutaten (zum Entfernen abwählen) ist jede Zutat mit einem Häkchen aktiviert. Häkchen entfernen = Zutat weglassen. Beispiel: das Häkchen bei „Zwiebeln" entfernen → auf dem Küchenzettel steht Ohne Zwiebeln. Der Preis bleibt gleich.

Bezahlte Extras hinzufügen

Unter den Extras setzen Sie ein Häkchen bei dem gewünschten Zusatz — z. B. Extra Käse · 2.000 IQD. Der Aufpreis wird automatisch zum Artikel addiert und erscheint auf dem Kundenbeleg.

✓ Alles per Häkchen

Zutaten und Extras werden angetippt, nicht getippt/geschrieben — das geht schnell und vermeidet Tippfehler. Welche Zutaten und welche Extras (mit Preis) ein Artikel hat, legt der Inhaber im Menü fest (Abschnitt 6).

04 Bestellungen — Nachdruck & Stornierung

Unter Bestellungen sehen Sie den Verlauf. Wählen Sie links eine Bestellung aus, um rechts alle Details zu sehen (Artikel, Zahlung, Uhrzeit, Kassierer).

AktionWas passiert
Beleg erneut druckenDruckt den Kundenbeleg noch einmal (z. B. wenn Papier gerissen ist).
Küchenzettel erneut druckenSchickt den Küchenzettel erneut an die Küche.
StornierenMacht eine falsche Bestellung ungültig. Sie bleibt sichtbar, aber als Storniert markiert und zählt nicht mehr zum Umsatz.

⚠ Stornieren nur mit Admin-PIN

Eine Stornierung verlangt eine Admin-PIN. So kann kein Kassierer allein Umsätze löschen. Ist bereits ein Admin angemeldet, geht es direkt.

05 Berichte & Tagesabschluss (Z-Report)

Unter Berichte sehen Sie auf einen Blick: Umsatz, Anzahl Bestellungen, durchschnittlicher Bestellwert und verkaufte Artikel — dazu Diagramme wie Top-Artikel, Umsatz nach Kategorie, Umsatz pro Stunde und Bar vs. Karte.

  • Zeitraum wählen: Heute, oder über Von / Bis einen eigenen Zeitraum eingeben und auf Anwenden tippen.
  • Kassierer sehen nur die Zahlen von heute. Den vollen Verlauf sieht nur der Admin.

Tagesabschluss (Z-Report)

Am Ende des Tages tippen Sie auf Z-Report erstellen. Das System fasst alle Verkäufe seit dem letzten Z-Report zusammen, speichert den Bericht und druckt ihn aus. Frühere Z-Reports finden Sie unter Frühere Z-Reports und können sie dort erneut drucken.

06 Menü verwalten

Unter Menü (nur Admin) pflegen Sie die komplette Speisekarte.

Artikel hinzufügen oder ausblenden

  • Artikel hinzufügen: Name, Kategorie und Preis (in IQD) eingeben, optional ein Foto hinzufügen.
  • Deaktivieren blendet einen Artikel aus der Kasse aus, ohne ihn zu löschen — z. B. wenn etwas ausverkauft ist. Mit Aktivieren kommt er zurück.

Kategorien hinzufügen oder löschen

Über Kategorien legen Sie neue Kategorien an (Kategoriename eingeben) oder entfernen bestehende mit Kategorie löschen.

Zutaten festlegen

Bei jedem Artikel tragen Sie unter Zutaten die Standardzutaten ein (tippen, dann Enter). Genau diese kann der Kassierer später an der Kasse per Häkchen weglassen (Abschnitt 3).

Extras festlegen (bezahlte Zusätze)

Unter Extras legen Sie bezahlte Zusätze mit Preis an — z. B. „Extra Käse" für 2.000 IQD. Diese kann der Kassierer pro Bestellung hinzufügen; der Preis wird automatisch zum Artikel addiert.

↺ Menü zurücksetzen

Auf Standardmenü zurücksetzen ersetzt die gesamte Karte durch das Thronburger-Standardmenü. Nur verwenden, wenn Sie wirklich neu anfangen wollen — Ihre eigenen Änderungen gehen dabei verloren.

07 Personal verwalten

Unter Personal (nur Admin) verwalten Sie Ihre Mitarbeiter.

  • Personal hinzufügen: Name, eine PIN (genau 4 Ziffern) und die Rolle (Admin oder Kassierer) festlegen.
  • PIN ändern: für jeden Mitarbeiter eine neue PIN vergeben — hiermit ändern Sie auch die Standard-PIN 1234 des Admins.

✓ Empfehlung

Jeder Mitarbeiter bekommt eine eigene PIN. So zeigen die Berichte korrekt Umsatz pro Kassierer und Sie sehen, wer welche Bestellung aufgenommen hat.

08 Einstellungen & Drucker

Unter Einstellungen (nur Admin) richten Sie die Drucker ein.

Drucker anschließen

  1. Schließen Sie den Bon-Drucker per USB an den PC an (oder im Netzwerk) und schalten Sie ihn ein. Windows erkennt ihn automatisch.
  2. Öffnen Sie Einstellungen → Druckereinstellungen.
  3. Wählen Sie bei Kassendrucker und Küchendrucker jeweils das richtige Gerät aus. Mit Systemstandarddrucker nimmt das System einfach den in Windows eingestellten Standarddrucker.
  4. Tippen Sie auf Testdruck — kommt eine Testseite, ist alles korrekt (Testseite gesendet). Zum Schluss Speichern.

Küchen- und Kassendrucker dürfen dasselbe Gerät sein (dann läuft alles über einen Drucker) oder zwei verschiedene (ein Drucker in der Küche, einer an der Kasse).

09 Datensicherung

Das System sichert Ihre Daten automatisch — Sie müssen nichts tun. Bei jedem Programmstart wird eine Sicherungskopie der Datenbank angelegt und die letzten 30 Kopien werden aufbewahrt; ältere werden automatisch entfernt.

✓ Empfehlung fürs Wochenende

Für maximale Sicherheit kopieren Sie den Sicherungsordner ab und zu auf einen USB-Stick. So sind Ihre Verkaufsdaten selbst bei einem Defekt des PCs geschützt. Fragen Sie hierzu einmalig den Techniker, wo genau der Sicherungsordner auf diesem PC liegt.

10 Hilfe bei Problemen

ProblemLösung
Meldung „Gespeichert, aber Druck fehlgeschlagen" Die Bestellung ist sicher gespeichert. Drucker prüfen (Strom, Papier, Kabel), dann in Bestellungen die Bestellung wählen und Beleg erneut drucken.
Kein Beleg kommt heraus Einstellungen → Testdruck. Kommt nichts, den richtigen Drucker auswählen und speichern.
PIN vergessen Ein Admin setzt die PIN unter Personal → PIN ändern neu.
Falsche Bestellung abgeschlossen Bestellungen öffnen, Bestellung wählen, Stornieren (Admin-PIN nötig), dann neu aufnehmen.
Programm reagiert nicht Programm schließen und neu öffnen. Es gehen keine Daten verloren — alles ist gespeichert.
